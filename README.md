# 🚀 מערכת ניהול Full Stack

ברוכים הבאים למערכת ניהול Full Stack!

פרויקט זה הוא אפליקציית רשת מקיפה לניהול משתמשים ותוכן, אשר פותחה באמצעות **React**, **Node.js** ו-**MySQL**.

המערכת כוללת מנגנון אימות משתמשים מאובטח, פאנל ניהול ייעודי למנהלים, וביצועים אופטימליים לטיפול בכמויות מידע גדולות.

---

## ✨ תכונות מרכזיות

### 👨‍💼 פאנל ניהול (Admin Dashboard)

- גישה בלעדית למנהלי המערכת.
- צפייה וניהול של כלל משתמשי המערכת.
- עריכה מאובטחת של פרטי משתמשים.
- איפוס ודריסת סיסמאות בעת הצורך.
- חסימה ושחרור חשבונות באמצעות מנגנון **Soft Delete / Soft Block**.

### ⚡ ביצועים ואופטימיזציה

- **סינון וחיפוש בצד הלקוח** - הפחתת עומס על השרת וצמצום מספר בקשות ה-API.
- **Pagination בצד השרת ובמסד הנתונים** עבור אלבומים ונתונים גדולים.
- טעינה יעילה של נתונים ושיפור חוויית המשתמש.

### 🗄️ מסד נתונים רלציוני

- שימוש ב-**MySQL** עם מבנה נתונים מסודר.
- הגדרת **Foreign Keys** לשמירה על שלמות הנתונים.
- תמיכה במנגנון **Soft Delete** באמצעות השדה `is_deleted`.

### 🔒 אבטחה והרשאות

- **Protected Routes** בצד הלקוח.
- בקרת גישה מבוססת הרשאות בצד השרת.
- גישה למידע רגיש מתאפשרת רק למשתמשים מורשים.

## 📁 מבנה הפרויקט

```text
project/
│
├── client/          # אפליקציית React
│
├── server/          # שרת Node.js ו-Express
│
└── database/
    └── schema.sql   # סקריפט יצירת מסד הנתונים
```

---

## ⚙️ התקנה והרצה

### 1️⃣ הגדרת מסד הנתונים

1. פתח תוכנת ניהול MySQL (לדוגמה: **DBeaver** או **MySQL Workbench**).
2. צור מסד נתונים חדש עבור הפרויקט.
3. הרץ את קובץ הסקריפט:

```text
database/schema.sql
```

הסקריפט ייצור באופן אוטומטי:

- טבלאות.
- קשרים בין הטבלאות.
- מפתחות זרים.
- הגדרות נוספות הנדרשות למערכת.

## 🚀 הפעלת צד השרת (Backend)

נווט לתיקיית השרת:

```bash
cd server
```

התקן את התלויות:

```bash
npm install
```

הפעל את השרת:

```bash
npm run dev
```

---

## 🎨 הפעלת צד הלקוח (Frontend)

פתח טרמינל נוסף ונווט לתיקיית הלקוח:

```bash
cd client
```

התקן את התלויות:

```bash
npm install
```

הפעל את סביבת הפיתוח:

```bash
npm run dev
```

## 🔑 חשבון מנהל (לצורך בדיקות)

ניתן להתחבר לפאנל הניהול באמצעות החשבון הבא:

| שדה | ערך |
|------|------|
| אימייל | `test@test.com` |
| סיסמה | `1234` |

> **הערה:** משתמש זה מוגדר במסד הנתונים עם ההרשאה:
>
> ```sql
> is_admin = 1
> ```
